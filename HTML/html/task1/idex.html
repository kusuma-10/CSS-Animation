What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to structure web content like text, images, and links.

HTML works alongside CSS and JavaScript to build beautiful and interactive websites.

How Does HTML Work?

HTML files are saved with the .html extension. These files contain tags that browsers interpret and display visually.

Learn more: HTML Introduction

'Hello World' Page in HTML

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <p>Hello World</p>
</body>
</html>
